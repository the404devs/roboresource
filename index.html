Owen's Robotics Resource Collection!

I've been involved in Lego Robotics since 2009, and have accumulated plenty of resources over the years, so I've decided to compile them in one place for future reference.

  • 2009 - 2012: Robotics Team (York Regionals) @ Oscar Peterson PS, using NXTs.
  • 2010 - 2014: Cybercamp (as camper), using NXTs.
  • 2014 - 2018: Cybercamp (as counsellor), using EV3s.
  • 2015 - 2022: Whitchurch-Stouffville Public Library, teaching EV3 and WeDo robotics classes (as volunteer 2015-2018, worked part-time Summer 2019 & Summer 2022).
  • 2021 - 2024: Idea Lab Kids Stouffville, running both junior WeDo and senior EV3 classes.
